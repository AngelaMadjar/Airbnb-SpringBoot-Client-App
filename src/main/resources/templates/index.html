This is a simple Spring Boot web application coded in Java. It enables CRUD operations on listed AirBnb accommodations persisting in a PostgreSQL database. Furthermore, it communicates with a Price Predictor microservice that estimates the price per night of an accommodation depending on its characteristics and features. The Price Predictor is a Linear Regression Machine Learning model deployed as a REST API in a Flask application.
